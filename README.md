## About this repository

This repository contains all coding activities, exercises, macros, and example scripts developed during the **DUNE-TECH Boot Camp at Rice University**. The boot camp is designed for undergraduates exploring research activities within the [DUNE experiment](https://www.dunescience.org/), providing practical training in the computational tools used daily in high-energy physics research.

Each module is self-contained with exercises, reference scripts, and notes.

### ROOT
The [ROOT framework](https://root.cern) — CERN's data analysis ecosystem 

---

### GEANT4 — Digital Twin
[GEANT4](https://geant4.org) is a Monte Carlo toolkit for simulating the passage of particles through matter. This helps to generate a **digital twin** of the detector, a full virtual replica of the physics environment used to validate reconstruction algorithms and optimize detector design.

---

### Machine Learning
Practical machine learning for particle physics — from classical methods to modern deep learning, applied to DUNE-relevant problems such as event classification and reconstruction.
